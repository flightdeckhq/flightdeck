# Flightdeck smoke test dependencies.
#
# Install into the venv used to run the smoke suite:
#
#     pip install -e sensor/
#     pip install -r tests/smoke/requirements.txt
#
# Or via the Makefile target:
#
#     make test-smoke-deps    # install these
#     make test-smoke         # install + run
#
# Versions are pinned with >= lower bounds (not exact) so local dev
# environments tolerate compatible upgrades. Only pin tighter if a
# regression in a newer release requires it.

# because Groups 1-11 exercise them directly).
